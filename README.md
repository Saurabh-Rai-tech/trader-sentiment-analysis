# 📊 Trader Performance vs Market Sentiment Analysis

## 📌 Objective

This project analyzes how market sentiment (Fear vs Greed) affects trader behavior, performance, and risk-taking.

---

## 📂 Datasets Used

1. **Bitcoin Market Sentiment Dataset**
2. **Historical Trader Data (Hyperliquid)**

---

## ⚙️ Methodology

* Data cleaning and preprocessing
* Handling missing values and column normalization
* Merging datasets using date
* Feature engineering (PnL, leverage, trade size)
* Exploratory Data Analysis (EDA)
* Visualization using charts

---

## 📊 Key Analysis

### 🔹 1. Fear vs Greed Performance

* Compared average PnL across sentiment categories
* Found better performance during Greed periods

### 🔹 2. Trading Behavior

* Trade size and frequency vary with sentiment
* Fear → less trading, Greed → aggressive trading

### 🔹 3. Leverage Impact

* High leverage → high risk & high volatility
* Low leverage → stable returns

### 🔹 4. Trade Direction

* Greed → more long positions
* Fear → more short positions

## 💡 Key Insights

* Traders perform better during Greed periods
* Fear leads to reduced trading activity and higher losses
* High leverage increases both profit and risk
* Trading behavior is strongly influenced by market sentiment

---

## 🚀 Strategy Recommendations

* Reduce leverage during Fear periods
* Avoid overtrading during Greed
* Focus on consistent and risk-managed strategies
